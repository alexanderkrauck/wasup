Support

Wasup ist ein Ein-Personen-Projekt — der schnellste Weg ist eine E-Mail an alexander.krauck@gmail.com. Antwort normalerweise innerhalb von 48 Stunden.

Häufige Anliegen

  • Falsche oder veraltete Event-Daten: Link zum Event (oder zur Quelle) mitschicken — wird korrigiert.
  • Veranstalter: Daten entfernen lassen (Takedown): formlose E-Mail genügt, Details in der Datenschutzerklärung.
  • Eine Quelle fehlt: URL der Veranstaltungsseite schicken — der Index nimmt neue Quellen laufend auf.
  • Probleme mit ChatGPT/Claude-Anbindung oder API: kurze Beschreibung + verwendete URL (https://wasup.at/mcp) mitschicken.
